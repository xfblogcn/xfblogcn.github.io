简介：C语言是一门面向过程的计算机编程语言，与C++、Java等面向对象编程语言有所不同。C语言的设计目标是提供一种能以简易的方式编译、处理低级存储器、仅产生少量的机器码以及不需要任何运行环境支持便能运行的编程语言。C语言描述问题比汇编语言迅速，工作量小、可读性好，易于调试、修改和移植，而代码质量与汇编语言相当。C语言一般只比汇编语言代码生成的目标程序效率低10%~20%。因此，C语言可以编写系统软件。

一、变量

  • 简介：C语言是一门强类型的编程语言！
  1. 标识符由字母，数字和下划线组成
  2. 第一个字符不能是数字，只能是字母或下划线
  3. 区分大小写
  4. C语言规定了一个标识符允许的字符个数（225），超过的字符将不被识别
  5. 八进制数字以0开头；十六进制数字以0x开头
  6. 变量一定要先定义后使用，并且定义后数据类型永不改变
  7. 一般变量要赋初值，否则其值是不确定的（初始化变量）

二、数据类型定义的返回值类型

1. 整数型变量的分类

  1. 基本型：int
  2. 短整型：short int(short)
  3. 长整型：long int(long)
  4. 无符号型：unsigned int，unsigned short，unsigned long ;// 只能保存不带符号的数字，不能存负数

运算符：sizeof() 看某个类型变量在计算机中占多少个字节，计算的是数据类型

  • 在一个常数后边加一个字母U，表示该常数用无符号整型方式存储，相当于unsigned int
  • 在一个常数后边加一个字母L，表示该常数用长整型方式存储，相当于1ong
  • 在一个常数后边加一个字母F，表示该常数用浮点方式存储，相当于float

2. 实型数据(浮点型数据)

  • float 单精度变量 一般提供7位有效数字(保守估计6位) 随机器而异
  • double 双精度变量 一般提供15-16位有效数字(保守估计15位) 随机器而异
  • 注意：存储变量是将十进制数转换为二进制数进行存储，我们用的时候是从内存中取出来又是将二进制数转换为十进制数进行读取，所以可能会遇到精度丢失问题！

科学计数法：168E-2 = 168 * 10 ^ (-2) = 1.68 // 浮点数在内存中都以指数形式存储

3. 字符型数据

  • 常规字符：用单引号引起来的一个字符
  • 特殊字符 => 转义字符：能够实现某种功能的特殊符号，如：换行\n.
  • 字符型常量：’a’,’\n’,’4’.
    • 说明：单引号引起来的一个字符，占一个字节
  • 字符型变量：char = ‘a’.
    • 说明：存储时实际是把字符对应的ascii码值存储到了内存中，范围（-128 – 127）
  • 字符串常量：”a”,”我爱你”,”hello world”.
    • 说明：双引号引起来的一堆字符，占两个字节(一个是本身，一个是\0)
    • 注意：\0 => 字符串结束标记，不输出，一般系统会隐式在字符串末尾添加\0

三、数值型数据之间的混合运算

  • 简介：运算对象即操作变量的类型不同时，较小的数据类型操作数会转换为较大的数据类型(范围更大)，然后再进行运算
  • 注意：所谓的“较大数据类型”，并不是说double类型实际上比int类型更大，而是说它还可以保存小数点之后的部分
  • 理解：不同的数据类型进行混合运算时，”系统会选取能表达最大数字的变量作为其他变量所转的目标类型进行”类型统一”，再运算
