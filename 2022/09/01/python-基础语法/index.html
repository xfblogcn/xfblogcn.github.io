Python简介：Python是由荷兰数学和计算机科学研究学会的吉多·范罗苏姆于1990年代初设计，作为一门叫做ABC语言的替代品。Python提供了高效的高级数据结构，还能简单有效地面向对象编程。Python语法和动态类型，以及解释型语言的本质，使它成为多数平台上写脚本和快速开发应用的编程语言，随着版本的不断更新和语言新功能的添加，逐渐被用于独立的、大型项目的开发。

注意：

  1. Python是一门强类型的编程语言
  2. Python中一切皆为对象
  3. Python语言是动态的
  4. Python在2022年至今编程语言热度排行榜第一！

一、关键字和标识符

1. 关键字

所谓关键字，就是python语言定义的、具有特殊用途的单词

  • 通过运行内置函数help()进入python交互式命令行帮助模式，使用keywords变量输出所有关键字

  • 通过导入模块keyword查看所有关键字

2. 标识符

所谓标识符，就是给程序中的变量、函数、方法、类等命名的名字

  • 标识符的组成：数字、字母、下划线

  • 标识符的命名规则：

    1. 区分大小写
    2. 不能是关键字
    3. 不能以数字开头
    4. 不能包含空格、制表符、数字符号、中划线、箭头

  • 标识符的命名规范：

    1. 见名知意
    2. 所有单词全部小写，单词之间用下划线进行分隔

二、数据类型

获取数据的数据类型：调用内置函数type可以获取数据的数据类型

1. 整数类型

  • 整数的进制：十进制默认的进制、二进制以0b开头、八进制以0o开头、十六进制以0x开头

  • 整数转换为不同进制的字符串

    • bin()：将十进制整数转换为2进制（binary）字符串

    • oct()：将十进制整数转换为8进制（octal）字符串

    • hex()：将十进制整数转换为16进制（hexadecimal）字符串

  • 整数的创建

    • 既可以直接创建一个整数，还可以调用内置函数int创建整数

    • 不传递任何参数时，返回整数0

    • 只传递一个参数时，将传递的参数转换为整数

    • 传递两个参数时，第一个参数必须是字符串，第二个参数指定几进制

2. 浮点数类型

  • 浮点数的创建

    • 既可以使用小数创建浮点数，还可以调用内置函数float创建浮点数

    • 不传递任何参数时，返回浮点数0.0

    • 只传递一个参数时，将传递的参数转换为浮点数

  • 用科学计数法表示浮点数

    • 很大或很小的浮点数可以用科学计数法来表示：men –> m乘以10的n次方

  • 浮点数存储的不精确性

    注意：计算机采用二进制存储浮点数时是不精确的，可能会存在误差，因此对于浮点数的运算需要格外小心

3. 布尔类型

  • 布尔类型只有两种取值：True、Flase（首字母必须大写）

  • 在进行数学运算时，True会被视为整数1，False会被视为整数0（bool类是int类的子类）

4. 对象None

注意：对象None是占据一定的内存空间的，它并不意味着“空”或“没有定义”

  • 对象None的使用场景

    • 用于变量的初始化

    • 将定义过的变量重置为”数据值不存在“的状态

5. 数据类型之间的转换

  • python中数值类型不能直接和字符串使用+进行拼接，不存在隐性转换，需要进行类型转换

注意：C语言中类型转换语法为**(类型名)变量：(int)a，而Python中则是使用函数转换**：int(a)

三、运算符

1. 算术运算符

  • 运算符：加+、减-、乘*、除/、整除//、取余%、幂**

**注意：python不支持自增++和自减–**，可使用i+=1的方式

2. 布尔运算符

  • 运算符：（python中布尔运算符只支持关键字不支持符号）

    • and：当两个运算符都为True时，运算结果才为True

    • or：只要有一个运算符为True，运算结果就为True

    • not：如果运算数为True（False），运算结果就为False（True）

注意：逻辑运算结果不一定只是True或者False，还有可能是数值。此时，谁决定结果则返回谁的值

3. 赋值运算符

  • 运算符：=、+=、-=、*=、/=、%=、**=

  • 重点：赋值运算符支持链式赋值

在Python中，变量是没有固定的数据类型的，但是它们引用的对象是有数据类型的

这是因为python是一种动态类型语言，这意味着变量可以随时引用不同类型的对象。变量的灵活性使得Python非常便于使用，但在操作变量之前，我们通常需要确保它们引用了我们所期望的对象类型，以避免出现错误。这就是为什么在Python中经常使用 type() 函数来检查对象的类型，以及为什么要遵循适当的编程实践，以确保变量的数据类型正确。

4. 比较运算符

  • 运算符：<、<=、>、>=、==、!=、is、is not

    • ==：用于比较两个运算数是否相等，主要用于“相等性”测试。简单的说，==判断值是否相等

    • is：用于比较两个运算数是否是同一个对象，主要用于“同一性”测试。简单的说，is判断地址是否相等

  • 不可变类型对象的is比较

    注意：对于不可变类型的对象，其内存可能会被重用，比如数值较小的整数对象

5. 条件运算符

注意：Python中不支持三目运算符 ? :

  • 运算符：’True’ if 条件 else ‘False’

6. 成员运算符

  • 运算符：in、not in（返回布尔值）

7. 位运算符

  • 运算符：与&、或、非~、异或^、左移<<、右移>>
    • &：两位都是1才为1，否则为0
    • ：只要有一位为1，则为1，否则为0
    • ~：如果为1，则为0，如果为0，则为1
    • ^：如果两位相同，则为0，不同为1
    • <<：左移数值扩大两倍
    • >>：右移数值缩小两倍

8. 运算符的优先级和结合性

  • 对于包含多个运算符的复杂表达式，可读性较低，为了提高可读性，建议的做法：
    • 在复杂表达式中使用小括号指定运算顺序
    • 将复杂表达式拆分成几步来完成

四、序列类型range

range是一种序列类型，用于表示不可变的整数序列，其返回值是一个可迭代对象

优点：不管range对象表示的整数序列有多长，所有range对象占用的内存空间都是相同的，因为仅仅需要存储start、stop和step。只有当用到range对象时，才会去计算序列中的相关元素

注意：range对象虽然不是生成器，但它支持惰性运算，在需要时按需生成序列中的元素，而不会一次性占用大量的内存空间

  • 可以调用内置函数range（类range的构造方法）创建range类型的对象（左闭右开）：
    • range(stop)
    • range(strat, stop)
    • range(strat, stop, step)

五、列表

列表的特点：

  1. 列表中的所有数据都有两个整数类型的索引，通过指定的索引总能映射到唯一确定的数据
  2. 列表中可以保存任何类型的数据，多种类型的数据可以混合存储在一个列表中（但不建议）
  3. 列表可以根据需要动态地伸缩，系统会根据需要动态地分配和回收内存，因此，在使用前无须预先声明列表的容量

列表相当于其他编程语言中的数组，是有序的、可重复的、可变的！

1. 列表的创建

  1. 使用中括号

  2. 调用内置函数list（类list的构造方法）

2. 列表的查操作

  1. 获得指定元素的索引，语法：L.index(val, start, stop)（左闭右开）

    • 方法index返回的索引值是大于零的

    • 若列表中存在多个指定元素，方法index只返回第1个指定元素的索引值

    • 若列表中不存在指定的元素，方法index抛出ValueError异常

  2. 通过索引获取元素，语法：L[index]（一次只能获取一个元素）

  3. 通过切片获取元素，语法：L[start:stop:step]（一次可以获取多个元素，左闭右开）

    • 切片得到的仍然是列表，是原始列表的片段的一份拷贝

    • 语法格式中三个参数均可以根据要求省略，step为步长，可以为负数，为负数时即从后往前获取元素

    • 切片操作是允许索引越界的

    关于内置函数slice（类slice的构造方法）创建slice类型的对象，三种调用方式：

    1. slice(stop)
    2. slice(start, stop)
    3. slice(start, stop, step)

    其中：start、stop和默认值都是None，但均不可省略，只有step的值可以省略

    注意：slice(start, stop, step) 与 start:stop:step是等价的

3. 列表的改操作

  1. 为指定索引的元素赋予一个新值（一次只修改一个元素）

  2. 为指定的切片赋予一个新值（一次至少修改一个元素）

4. 列表的增操作

  1. 调用方法append（在列表的末尾一次只添加一个元素）

  2. 调用方法extend（在列表的末尾一次至少添加一个元素）

  3. 调用方法insert（在列表的任意索引位置一次只添加一个元素）

  4. 为指定的切片赋予一个新值（在列表的任意索引位置一次至少添加一个元素）

5. 列表的删操作

  1. 调用方法remove（一次只删除一个指定的元素）

    • 存在多个指定元素，则只删除第一个

    • 不存在需要删除的指定元素，则抛出异常

  2. 调用方法pop（一次只删除一个指定索引的元素）

    • 该方法返回被删除的元素

    • 指定的索引不存在，则抛出异常

    • 不指定索引，则默认删除列表的最后一个元素

  3. 使用del语句（一次至少删除一个元素）

  4. 为指定的切片赋值一个空列表（一次至少删除一个元素）

  5. 调用方法clear清空列表

6. 使用运算符操作列表

  1. 使用加法运算符操作列表

    • 将两个列表合并后生成一个新列表，被合并的两个列表不发生变化

    • 参数赋值运算符+=会对列表本身进行修改

  2. 使用乘法运算符操作列表

    • 将列表中的所有元素重复n次后生成一个新列表，被乘的列表不发生变化

    • 参数赋值运算符*=会对列表本身进行修改

  3. 使用比较运算符比较两个列表

    • 比较规则：依次比较两个列表中的每一个元素，相等则继续，不等则得出结果，且后续元素将不再被比较

7. 列表的反转

  1. 调用方法reverse，无返回值，直接改变列表本身

  2. 调用内置函数reversed，返回值是一个迭代器对象，且列表本身不会被改变

8. 列表的排序

  1. 调用方法sort，无返回值，直接改变列表本身

    • 默认按照从小到大排序，可以指定参数reverse = True，从而按照逆序进行排序

  2. 调用内置函数sorted，返回一个新的排好序的列表，且列表本身不会被改变

内置函数sorted延伸：

作用：sorted()函数的作用是对可迭代对象（例如列表、元组、字符串、集合等）进行排序，并返回一个新的排好序的列表（无论是对什么类型的数据使用sorted函数排序，返回的都一定是列表）。它不会修改原始可迭代对象，而是返回一个新的有序列表

语法：sorted(iterable, key=None, reverse=False)

9. 多维列表

  • 一维列表的操作也适用于多维列表

  • 多维列表的初始化

六、元组

为什么设计元组？

  • 由于元组是不可变类型的对象，对于不可变类型的对象，在多任务环境下同时操作对象时不需要加锁。因此，在程序中应尽量使用不可变类型的对象

元组是有序的、可重复的、不可变的（但存储的可变类型数据对象是可改其值而不可改其引用的）！

1. 元组与列表的区别

  • 元组用小括号表示（且小括号是可省略的），列表用中括号表示

  • 元组是不可变类型，列表是可变类型

    • 对于元组中可变类型的数据，元组中储存的是其引用（在内存中的地址），即引用不能被改变，但引用所指向的可变类型的数据值是可以被改变的

2. 只包含一个元素的元组

  • 元组中至少要包含一个逗号，即使元组只有一个元素，否则，小括号会被看做是数学公式中的小括号

3. 解构赋值

  • 使用赋值运算符实现多个变量同时赋值

    • 赋值运算符的左边是一个所有元素都为变量的元组或列表

    • 赋值运算符左右两边的元素个数必须是相同的

  • 可以在赋值运算符左边的某个变量前加*，以匹配赋值运算符右边的0个或多个元素

  • 交换两个变量的值

七、字符串

字符串的特点：

  1. 字符串也是序列类型，是字符的有序集合
  2. 对字符串进行修改时会开辟一块新的内存空间，其内存地址也会发生改变
  3. 字符串的单引号和双引号都无法取消特殊字符的含义
  4. python没有单独的字符类型，字符就是只包含一个元素的字符串

字符串是有序的、可重复的、不可变的！

1. 字符串的创建

  1. 使用单引号或双引号直接创建，一般使用单引号（引号可以嵌套，可以在单引号中嵌套双引号，反之也可以）

  2. 调用内置函数str（类str的构造方法）

2. 原始字符串

  • 可以在字符串前面添加r或R使其转义字符失效，从而将字符串声明为原始字符串

    • 注意1：原始字符串的最后一个字符不能是反斜杠

    • 注意2：最后两个字符都是反斜杠代表原始字符串结束，在注意1中除外

3. 使用运算符操作字符串

  1. 使用加法运算符操作字符串

    • 可以使用加法运算符将两个字符串连接后生成一个新字符串

    • 将多个字符串常量放在一起就会自动连接然后生成一个新字符串

    • 将多个字符串常量写在多行中，可以在每行的末尾添加一个反斜杠

  2. 使用乘法运算符操作字符串

    • 可以使用乘法运算符将字符串中的所有字符重复n次后生成一个新字符串

  3. 使用比较运算符比较两个字符串

    • 两个字符进行比较时，比较的是其ordinal value（ASCll码值）

    • 调用内置函数ord可以得到指定字符的ordinal value

    • 调用内置函数chr时指定ordinal value可以得到其对应的字符

    注意：同数值较小的整数对象一样，字符串常量也会被缓存和重用，以至于is比较时即使不是同一引用也会返回True

4. 字符串的查操作

  • 可以调用方法index、find、rindex、rfind

    • 当字符串中存在多个被查找的子串时，方法index和find返回第一个子串的索引， 方法rindex和rfind返回最后一个子串的索引

    • 当字符串中不存在指定的子串时，方法index和rindex抛出ValueError，方法find和rfind返回-1

    此外，字符串没有”改”操作、”增”操作和”删”操作，因为字符串是不可变类型

    如果想要得到需要字符串，可以使用切片操作后配合+号将其拼接起来，从而得到需要的字符串

5. 字符串的反转与排序

  1. 字符串的反转

    • 由于字符串是不可变类型，所以不存在方法reverse，只能调用内置函数reversed（字符串本身不会被改变）

  2. 字符串的排序

    • 由于字符串是不可变类型，所以不存在方法sort，只能调用内置函数sorted（字符串本身不会被改变），且默认按照ascll排序的

    • 自定义排序规则：调用内置函数sorted时，还可以指定参数key=函数名或key=类名.方法名，这样会对字符串中的所有字符分别调用指定的函数或方法，然后按照函数或方法的返回值进行排序

      注意：参数key同样适用于列表和元组，且调用列表的方法sort时，也可以指定参数key

6. 使用百分号作为占位符

  • %作为占位符

    • %s：表示字符串，可以把任何数据类型转换为字符串

    • %d或者%i：表示整数

    • %f：表示浮点数

    • %%：将%转义为普通字符，即表示%

    • %Y-%m-%d %H:%M:%S：表示年月日 时分秒（注意区分大小写）

  • %作为占位符还可以指定宽度和精度，其中，数字和字符串都是右对齐

  • %作为运算符：当定义的格式化字符串中包含两个及两个以上的占位符时，所有的实际值必须封装在元组中

7. 使用花括号作为占位符

  • 可以调用方法format并使用花括号作为占位符，从而得到格式化字符串

    • 占位符{}中不指定参数，方法format的参数会按顺序依次匹配所有的占位符{}

    • 占位符{}中可以指定位置参数，0表示方法format的第1个参数，1表示方法format的第2个参数，以此类推

    • 占位符{}中可以指定关键字参数的名称，在方法format中对应指定关键字参数的名称和值

  • 占位符{}中可以使用冒号指定整数的表现形式，其中，位置参数和关键字参数的名称放在冒号前面

  • 占位符{}中也可以使用冒号指定宽度和精度，其中，数字是右对齐，字符串是左对齐

  • 占位符{}中还可以使用冒号指定其他格式

还可以调用内置函数format得到格式化字符串，它与字符串的format方法是等价的

8. 使用美元符作为占位符

  • 可以导入模块string中的类Template并使用美元符$作为占位符，从而得到格式化字符串

    1. 创建Template实例

    2. 调用substitute（或safe_substitute）方法。其中，substitute当占位符没有匹配的实际值时，抛出KeyError；而safe_substitute当占位符没有匹配的实际值时，不会抛出KeyError，而是直接输出占位符

9. 字符串常用方法

  • 字符串的大小写转换

    • upper：把所有字符全部转换为大写

    • isupper：判断所有字符是否全为大写

    • lower：把所有字符全部转换为小写

    • islower：判断所有字符是否全为小写

    • title：把每个单词的第一个字符转换为大写，其余字符转换为小写

    • istitle：判断每个单词的第一个字符是否为大写并且每个单词的剩余字符为小写

    • swapcase：把所有小写字符转换为大写，所有大写字符转换为小写

    • capitalize：把整个字符串的第一个字符转换为大写，其余字符转换为小写

  • 字符串的对齐

    • center：中心对齐，第一个参数指定字符串的宽度，第二个参数指定填充字符

    • ljust：左对齐，第一个参数指定字符串的宽度，第二个参数指定填充字符

    • rjust：右对齐，第一个参数指定字符串的宽度，第二个参数指定填充字符

    • zfill：右对齐，左边用0填充，仅一个参数，用于指定字符串的宽度

    • 注意：如果指定的宽度小于等于字符串的长度，则返回字符串本身；且默认填充字符为空格，是可选的

  • 字符串的子串替换

    • replace：将字符串中的某个子串替换为指定的字符串

    • 使用：replace可以接受3个参数，参数1指定被替换的子串，参数2指定替换子串的字符串，参数3指定最大替换次数，是可选的

    • 注意：该方法返回替换后得到的字符串，替换前的字符串不发生变化

  • 字符串的字符转换

    • maketrans+translate：对字符串中的某些字符进行字符转换，通常用于执行字符替换或删除操作

    • 使用：**str.maketrans()方法用于创建一个字符映射表，该方法接受3个参数，参数1与参数2长度必须相同，用于指定源字符和目标字符的映射关系；参数3用于指定字符串中需要删除的字符，是可选的。text.translate()**方法用于执行实际的字符转换或删除操作。

  • 字符串的劈分与合并

    • split：从字符串的左边开始劈分，默认的劈分符是空格字符，返回值是一个列表

    • rsplit：从字符串的右边开始劈分，默认的劈分符是空格字符，返回值是一个列表

    • 参数sep：用于指定劈分符

    • 参数maxsplit：用于指定最大劈分次数，在经过最大次劈分后，剩余的子串会单独作为一部分

    • 注意：split和rsplit的结果的索引都是相同的，它们的区别仅在于分割的顺序不同

    • partition：从字符串的左边第一个劈分符开始劈分，调用时必须指定劈分符，返回值是一个元组

    • rpartition：从字符串的右边第一个劈分符开始劈分，调用时必须指定劈分符，返回值是一个元组

    方法partition和rpartition的作用是将字符串劈固定分为三部分：

    1. 劈分符前面的部分
    2. 劈分符
    3. 劈分符后面的部分
    • 调用方法join合并多个字符串

  • 以is开头的字符串方法

    • isidentifier：判断指定的字符是否是合法的标识符

    • iskeyword：判断一个字符串是否是关键字（模块keyword中的方法iskeyword）

    • isspace：判断指定字符串是否全部由空白字符组成（\t、\r、 、\n都属于空白字符）

    • isalpha：判断指定字符串是否全部由字母组成

    • isdecimal：判断指定字符串是否全部由十进制的数字组成

    • isnumeric：判断指定字符串是否全部由数字组成

    • isalnum：判断指定字符串是否全部由字母和数字组成

  • 去除字符串的前导字符串或后续字符串

    • lstrip：去除字符串的前导字符串，默认为空格字符串

    • rstrip：去除字符串的后续字符串，默认为空格字符串

    • strip：去除字符串的前导字符串和后续字符串，默认均为空格字符串

    调用以上三个方法时可以指定1个字符串： 对于lstrip：从左边第1个字符开始依次往后，直到某个字符不在指定的字符串中 对于rstrip：从右边第1个字符开始依次往前，直到某个字符不再指定的字符串中

八、字典

字典的特点：

  1. 字典可以根据需要动态地伸缩，系统会根据需要动态地分配和回收内存，因此在使用前无须预先声明字典的容量
  2. 字典会浪费较大的内存，与列表相比，是用空间换取了时间

字典是无序的、不可重复的（key不可重复，value可以重复）、可变的（key必须是不可变对象）！

1. 字典的创建

  1. 使用花括号

  2. 使用内置函数dict（类dict的构造方法）

  3. 调用dict的方法fromKeys

    • 调用该方法时通过参数指定所有的key，所有value的默认值都为None，可以通过参数统一指定所有value的值

为字典中指定的key设置默认的value值，调用方法setdefault

目的：为了确保字典中指定的key总是存在的

用法：如果指定的key不存在于字典中，它会自动创建该键，并将其对应的值设置为提供的默认值。如果指定的键已经存在于字典中，则不会进行任何更改，并返回该键对应的值

2. 字典的查操作

  1. 使用中括号

  2. 调用方法get

  3. 此外，可以使用运算符in（not in）检查字典中是否存在（不存在）指定的key

3. 字典的改操作

  1. 为已经存在的key赋予一个新的value值（一次只修改一个key对应的value）

  2. 调用方法update（一次至少修改一个key对应的value）

4. 字典的增操作

  1. 为不存在的key赋予一个value值（一次只添加一个key-value对）

  2. 调用方法update（一次至少添加一个key-value对）

5. 字典的删操作

  1. 调用方法pop，返回值为被删除的value（一次只删除一个指定key的key-value对）

  2. 使用del语句（一次只删除一个指定key的key-value对）

  3. 调用方法popitem（一次只删除一任意的key-value对）

  4. 调用方法clear（清空字典）

6. 字典的视图

  • 得到字典相关视图的三个办法（视图会同步字典的变化）：

    1. Keys：返回字典所有key的视图

    2. Values：返回字典所有value的视图

    3. Items：返回字典所有key-value对的视图

7. 借助字典创建格式化字符串

  1. 使用百分号作为占位符

    • 解释：当格式化字符串中的占位符的值来自于某个字典的value时，可以在占位符中添加一对括号，即**(字典的key)，在百分号后面直接使用字典名完成赋值（注意，括号中的值会被隐式添加一对引号，因此，如果字典中key是字符串，则在括号中需要去掉key自带的引号**）

  2. 使用花括号作为占位符

    • 与使用百分号作为占位符同理，但需要调用方法format_map并把该字典直接作为方法的参数

九、集合

集合的特点：

  1. 集合，简而言之，即没有value的字典
  2. 集合存储的数据一定是不可变类型的，所以只有增删查操作，而没有改操作
  3. 集合可以根据需要动态地伸缩，系统会根据需要动态地分配和回收内存，因此在使用前无须预先声明集合的容量
  4. 集合不包含重复元素，内存占用相对较小，适合用于去重操作和快速判断元素是否存在

集合是无序的、不可重复的（重复则自动剔除）、可变的！

1. 集合的创建

  1. 使用花括号

  2. 调用内置函数set（类set的构造方法）

    对list去重的小技巧：调用set()函数将list列表转换为set集合

2. 集合间的关系

  1. 两个集合是否相等（使用运算符==和!=）

  2. 一个集合是否是另一个集合的子集（调用方法issubset）

  3. 一个集合是否是另一个集合的超集（调用方法issuperset）

  4. 两个集合是否没有交集（调用方法isdisjoint）

3. 集合的数学操作

  1. 交集（调用方法intersection或使用运算符&）

    • 做交集操作后生成一个新集合，做交集操作的两个集合不变

    • 也可以调用方法intersection_update，其返回值为None，但会使用intersection的返回值（交集操作后生成的新集合）更新调用者

  2. 并集（调用方法union或使用运算符）

    • 做并集操作后生成一个新集合，做并集操作的两个集合不变

    • 不存在方法union_update

  3. 差集（调用方法difference或使用运算符-）

    • 做差集操作后生成一个新集合，做差集操作的两个集合不变

    • 也可以调用方法difference_update，其返回值为None，但会使用difference的返回值（差集操作后生成的新集合）更新调用者

  4. 对称差集（调用方法symmetric_difference或使用运算符^）

    • 做对称差集操作后生成一个新集合，做对称差集操作的两个集合不变

    • 也可以调用方法symmetric_difference_update，其返回值为None，但会使用symmetric_difference的返回值更新调用者

4. 集合的查操作

  • 可以使用运算符in（not in）检查集合中是否存在（不存在）指定的元素

5. 集合的增操作

  1. 调用方法add（一次只添加一个元素）

  2. 调用方法update（一次至少添加一个元素）

6. 集合的删操作

  1. 调用方法remove（一次只删除一个指定的元素）

  2. 调用方法discard（一次只删除一个指定的元素）

  3. 调用方法pop（一次只删除一个任意的元素）

  4. 调用方法clear（清空集合）

7. 不可变集合frozenset

  • 可变性：不可变集合 (frozenset) 是不可变的数据结构，一旦创建后就不能再修改，不能添加、删除或更新元素（frozenset之于set就好比tuple之于list）

  • 哈希性：不可变集合 (frozenset) 是可哈希的，因为它是不可变的，可以作为字典的key或其他集合中的元素（集合中存储的数据一定是不可变类型的）

  • frozenset对象的创建：调用内置函数frozenset（类frozenset的构造方法）

十、流程控制

任何简单或复杂的算法都可以由顺序结构、选择结构和循环结构这三种基本结构组合而成

  • 顺序结构：程序从上到下顺序地执行代码，中间没有任何判断和跳转，直到程序结束
  • 选择结构：程序根据判断条件的布尔值选择性地执行部分代码
  • 循环结构：程序根据循环条件反复执行某段代码，直到不满足循环条件为止

1. 对象的布尔值

  • 任何对象都可被直接用作布尔值，解释器会自动调用内置函数bool进行转换

  • 以下对象的布尔值为Flase：False、数值零、None、空字符串、空列表、空元组、空字典、空集合

3. 条件表达式

  • 条件表达式是包含if-else语句的表达式，类似于C语言中的三目运算符

  • 语法格式：x if 判断条件 else y

  • 运算规则：如果判断条件的布尔值为True，则整个式子返回值为x；否则，返回值为y

4. while语句

  • while-True-break结构：通过while True构造一个无限循环，在循环体中满足某个条件时通过break退出循环

7. break-else结构

  • 作用：需要在循环正常结束后执行某些操作
  • 执行：循环中如果没有执行循环体中的break语句，即没有因为break提前退出循环，则会执行else从句

break与continue

  1. 在嵌套的循环语句中，break和continue默认作用于当前循环
  2. break用于结束整个循环，continue用于结束循环中的当前迭代，继续下一个迭代

十一、常用内置函数

1. 内置函数zip

  • 作用：并行遍历。将可迭代的对象作为参数，将对象中对应的元素打包成一个个元组，然后返回由这些元组组成的类

  • 注意：如果两个可迭代对象的长度不同，那么较长的会被截断；可以使用*对zip对象解压，返回二维矩阵式

2. 内置函数map

  • 作用：会使用指定的函数名作用于指定的可迭代对象的每个元素，然后生成新的可迭代对象

  • 参数：参数1指定函数名，参数2指定可迭代对象

3. 内置函数Filter

  • 作用：与内置函数map相似，会使用指定的函数名作用于指定的可迭代对象的每个元素，同时过滤掉函数返回值为False的相关元素，然后生成新的可迭代对象

  • 参数：参数1指定函数名，参数2指定可迭代对象

十二、生成式

由于累计基础知识在此阶段掌握并不足以理解元组生成式，所以以下暂时没有关于元组生成式的记录；

并在后续将其记录在了《02_Python_高级语法》的 五、生成器与迭代器 中。

1. 列表生成式

  • 语法格式：[ 表示列表元素的表达式 for 自定义的变量 in 可迭代对象 [if语句] ]

    • 其中，”表示列表元素的表达式”中通常包含”自定义的变量”

  • 使用规则：凡是可以通过for-in循环创建的列表，均可以使用列表生成式来创建

    • 列表生成式中可以使用双重循环

    • 列表生成式支持嵌套，即一个列表生成式中嵌套另一个列表生成式

2. 集合生成式

  • 语法格式：**{ 表示集合元素的表达式 for 自定义的变量 in 可迭代对象 [if语句] }**
    • 其中，”表示集合元素的表达式”中通常包含”自定义的变量”
    • 与列表生成式其余用法均相同，唯一的不同是使用花括号创建

3. 字典生成式

  • 语法格式：**{ 表示字典key的表达式: 表示字典value的表达式 for 自定义的表示key的变量, 自定义的表示value的变量 in 可迭代对象 [if语句] }**

    • 其中，”表示字典key的表达式”中通常包含”自定义的表示key的变量”，”表示字典value的表达式”中通常包含”自定义的表示value的变量”

    • 与列表生成式其余用法均相同，不同是使用花括号创建，且使用key:value表示
