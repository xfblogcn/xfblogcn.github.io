编译预处理（预编译）
C语言一般提供三种预处理功能：1、宏定义 2、文件包含 3、条件编译

一、宏定义

  • 简介：宏定义主要在预处理阶段完成，由预处理器进行字符替换，变量定义则会在内存中为变量分配地址
  • 定义：#define 标识符 字符串（其中标识符称为宏名，预编译时将宏名替换成字符串的过程称为宏展开，#define称为宏定义命令）
  • 作用：用一个指定的标识符来代表一个字符串，一般写在文件开头

不带参数的宏定义

  • 语法：#define 标识符 字符串
  • 好处：
    1. 用一个简单的名字替代一个很长的字符串
    2. 增加了修改的方便性，提高了程序的可移植性
  • 说明：
    1. 宏名一般都用大写字母
    2. 宏定义不是C语句，所以不用在末尾加分号，如果有分号会连分号一起替换
    3. #define命令出现在程序中函数的外面，宏名的有效范围是#define之后到本源文件结束，不能跨文件使用
    4. 可以用#undef命令终止宏定义的作用域
    5. #define用于宏定义时，还可以用于已经定义的宏，可以层层置换
    6. 字符串内的字符不进行替换（例：char stmp[100] = “DPICPI”; 即使与宏名DPICPI相同，也不进行替换）

带参数的宏定义

  • 语法：#define 宏名(参数表) 字符串
  • 说明：
    1. 一般字符串中都会包含参数中所指定的参数，但不包含或不包含完全也可以，只是参数没意义
    2. 宏定义时宏名和带参数的括号之间不能加括号，否则空格以后的字符都会作为替代字符串的一部分

宏定义和函数区别：

  • 函数调用是先求出实参表达式的值，然后带入形参，而宏定义只进行简单的字符替换，并不会求出表达式的值
  • 函数调用是在程序运行时处理，分配临时内存；而宏展开是在编译时进行的，不分配内存，也没有返回值、值传递等说法
  • 宏的参数没有类型这个说法，只是个符号，展开时带入指定字符串中
  • 使用宏次数多时，宏展开后源程序变长，函数调用就不会
  • 宏替换只占用编译时间，不占用运行时间；而函数调用占用的是运行时间(分配内存，传参，执行函数体，返回值)

三、文件包含

  • 简介：将另一个文件的内容包含到本文件中，通过#include实现
  • 语法：#include “文件名”
  • 用法：#include常用于公共头文件（即.h后缀文件）；常把宏定义、函数+外部全局变量声明等放在头文件中，需要时引用
  • 说明：
    1. 一个#include只能引入一个头文件，需要时引入多个#include即可
    2. 文件包含是可以嵌套的，头文件中还可以#include其他头文件
    3. #include <>是去系统目录中查找文件；#include “”是在当前目录中查找，找不到再到系统目录中查找

四、条件编译

  • 语法1：
    • 简介：当标识符被宏定义过(#define)时，则对程序段1进行编译，否则对程序段2进行编译；#else也可以没有
    • #ifdef 标识符
     程序段1(一堆代码)
#else
     程序段2
#endif
  • 语法3：
    • 简介：当指定表达式值为真(非0)时就编译程序段1，否则编译程序段2，这样就可以使程序在不同条件下实现不同的功能
    • #if 表达式
     程序段1(一堆代码)
#elif 表达式2
     程序段2
#else
     程序段3
#endif

条件编译的好处

  1. 条件编译可以减少生成的目标文件的长度，因为都是预编译时做的判断，实际行数就只剩执行文件那几行了，而用if else语句就是实打实的每一行
  2. 开发时跨平台使用，就必须用条件编译，增加程序的可移植性和灵活性
    • #if _WIN32
     WaitForSingleObject() //...windows上专用
#elif __linux__
     epoll()
#else
     ......
#endif

#include <stdio.h>
#include <stdlib.h>

// 宏定义
#define DEBUG //可以不要字符串，#ifdef也判断为真
// #define RELEASE //release : 发布、释放
#define BOOL 3>5

int main() //主函数
{
#ifdef DEBUG
printf(“调试程序执行的\n”);
#endif

#ifndef RELEASE
printf(“如果没有发布执行的\n”);
#endif

#if BOOL
printf(“BOOL为真输出程序段1\n”);
#else
printf(“BOOL为假输出程序段2\n”);
#endif

system("pause");
return 0;

}
