一、数组的指针和指向数组的指针变量

1. 指向数组元素的指针变量的定义和赋值

  • 语法：类型标识符 *标识符 = 数组名(或者&arr[0])
  • 说明：
    1. 数组指针就是数组的开始地址，数组元素的指针就是数组元素的地址
    2. 数组元素的内存一定是挨着的
    3. a = &a[0] = 开始地址，数组名代表的是数组的首地址！

2. 通过指针引用数组元素

  • 七个常用方法：条件 => int arr[9]; int *p = arr;
    1. *p = 19; //等价于arr[0] = 19;
    2. p = p + 1; //p的地址应该增加了4，因为指针是整型，即现在p应该指向了arr[1]
    3. p+i 或 arr+i //p+i或arr+i现在指向的就是a[i]的地址，也就是指向了数组arr的第i个元素
    4. *(P+i) 或 *(arr+i) //这两个是p+i或者a+i所指向的数组元素，也就是arr[i]
    5. p[i] //等价于a[i]等价于*(p+i)和*(a+i)
    6. *p++ //等价于*(p++)，p++是先用后加，所以*p++含义就是：先得到p指向的变量的值*p，然后再使p指针自加1，指向下一个数组元素
    7. *++p //等价于*(++p)，++p是先加后用，所以*++p含义就是：p指针先自加1，指向下一个数组元素，然后再得到p指向的变量的值
    8. (*p)++ //表示p所指向的元素的值加1，如果p指向的首地址，即a[0]++

注意：p+i或者a+i得到的都是地址，既然是地址，那就可以赋给指针变量

3. 函数名作为函数参数

  • 在函数中改变一个数组的元素的值，实参和形参的对应关系有4种：
    1. 实参和形参都是数组名，不涉及指针知识，即实参数组首地址和形参数组首地址共用一段内存，即两者是同一个数组
    2. 实参用数组名，形参用指针变量
    3. 实参和形参都用指针变量
    4. 实参用指针变量，形参用数组名(不常用)
      • 把指针传递给数组名，那这个数组名就相当于这个数组的首地址了；换―种理解方式：把形参数组名也当成指针看待，是一个指向数组a的首地址的指针

4. 指向多维数组的指针和指针变量探究

  • 注意：二维数组中一维数组名不占实际内存，只是一个地址
  • 定义：int a[3][4]
    1. 二维数组数组名代表的也是整个二维数组的首地址，即第0行的首地址
    2. 每一行的第一个元素又是二维元素每一行的首地址，首地址相差：每一行的元素个数*类型所占字节数
    3. a+1 a+2分别代表第一行首地址和第二行首地址，所以a+1应该跳过16个字节，a+2也是
    4. 二维数组中：a[0],a[1],a[2]是一维数组名；C语言规定数组名就代表数组的首地址；即：a[0] == &a[0][0] 是第0行的首地址
    5. 第0行第一列元素地址表示为&a[0][1]或者a[0]+1（a[0]+1就是一维数组名加1，跳过的是4个字节），两者等价

二维数组数据表 a 二维数组名，数组首地址，第0行首地址 是个地址:1000 a+1 第1行首地址 是个地址:1016 a+2 第2行首地址 是个地址:1032 a[0],&a[0][0],*a,*(a+0),&a[0] 第0行首地址(第0行第0列元素地址) 是个地址:1000 a[1],&a[1][0],*(a+1),&a[1] 第1行首地址(第1行第0列元素地址) 是个地址:1016 a[2],&a[2][0],*(a+2),&a[2] 第2行首地址(第2行第0列元素地址) 是个地址:1032 总结：*(a+i)等价于a[i]（是个地址） a[0]+1,&a[0][1],*a+1 第0行第1列元素地址 是个地址:1004 a[1]+2,&a[1][2],*(a+1)+2 第1行第2列元素地址 是个地址:1024 *(a[0]+1),a[0][1],*(*a+1) 第0行第1列元素值 是个值 *(a[1]+2),a[1][2],*(*(a+1)+2) 第1行第2列元素值 是个值

神奇的内容 a和*a，都是地址，而且地址值是相同的 a+1和*(a+1),a+2和*(a+2)也是同理，是个地址 a[i]和&a[i]都是地址，而且地址值也相同，是个地址（最终推论）

二、指针数组和数组指针总结与区别

1. 指针数组

  • 定义：int *P[10];（[]优先级 > *优先级）
  • 说明：指针数组中有10个元素，每个元素都是指针，这相当于定义了10个指针变量

2. 数组指针

  • 定义：int (*P)[10];
  • 说明：这是一个指针变量，这个指针变量用来指向含有10个元素的一维数组的首地址
