一、ubuntu-18.04.6安装

在m1芯片的MacBook上使用UTM虚拟机安装ubuntu-18.04.6（关键步骤）：

  1. 阿里云镜像站 ubuntu-18.04.6 下载链接：ubuntu-18.04.6-server-arm64.iso
  2. 一键配置系统源：wget http://fishros.com/install -O fishros && . fishros >> 5 >> 2 >> 0
  3. 安装ubuntu桌面：sudo apt install tasksel >> sudo tasksel install ubuntu-desktop
  4. 查询可用网卡：ifconfig -a
  5. 配置可用网卡：sudo vim /etc/netplan/01-netcfg.yaml >> netplan apply >> reboot
  6. 查看本机ip地址：ifconfig

二、MySQL安装配置

安装：sudo apt install mysql-server

  • 查看MySQL中的默认用户和密码：sudo cat /etc/mysql/debian.cnf（client配置为主）

  • 使用client配置中的user和password登录MySQL：mysql -udebian-sys-maint -pXXXXXX(密码)

    1. 选择mysql数据库：use mysql;
    2. 创建新用户xiaofu密码123456：CREATE USER 'xiaofu'@'localhost' IDENTIFIED BY '123456';
    3. 查看当前MySQL中所有的用户：select user,host from mysql.user;
    4. 赋予权限：grant all on*.*to xiaofu@localhost;

  • 配置远程访问权限：sudo vim /etc/mysql/mysql.conf.d/mysqld.cnf

    1. 注释代码：#bind-address = 127.0.0.1
    2. 重启mysql服务：sudo service mysql restart

三、Redis安装配置

安装：sudo apt install redis-server

  • 配置远程访问权限：sudo vim /etc/redis/redis.conf
    1. 注释代码：#bind 127.0.0.1 ::1
    2. 关闭保护模式：protected-mode no
    3. 重启redis服务：sudo service redis-server restart

四、MongoDB安装配置

安装：sudo apt install mongodb

  • 配置远程访问权限：sudo vim /etc/mongodb.conf
    1. 注释代码：#bind_ip = 127.0.0.1
    2. 重启mongodb服务：sudo service mongodb restart

五、miniconda安装配置

访问官网：https://docs.conda.io/projects/miniconda/en/latest/index.html 下载对应版本

安装miniconda：sh ./Miniconda3-latest-Linux-aarch64.sh

六、ssh远程连接配置

安装：sudo apt install openssh-server

  • ssh远程连接：

    • ubuntu启动ssh服务：sudo service ssh start

    • mac主机访问：ssh xiaofu@192.168.64.12
